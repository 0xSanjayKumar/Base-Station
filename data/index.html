How to Use This Base Station

GPS Coordinates

Enter the latitude, longitude, and altitude values. Set accuracy parameters and time as needed. Click 'Update' to apply or 'Stop' to cease updates.

Automatic Mode

For quick setup, use 'Auto Survey' to automatically collect position data or 'Auto Fix' to establish a fixed position based on known coordinates.

Manual Mode Survey

Use these controls for custom survey operations:

  • Reset: Clear all survey data
  • Port: Configure connection port
  • MSG-Enable: Toggle NMEA message output
  • Survey-Stat: View current survey statistics
  • Surveying: Begin manual survey process
  • Broadcast: Share position data
  • Status: Check system status

Manual Mode Fix

For precise positioning control, use these options to manually set up fixed-position operation.
